# Intro
Dever is a cross platform application designed to help developers with everyday task, provide overviews, and automate tedious tasks.

## Stack
Dever is built using:

Expo: https://docs.expo.dev/
Atlassian Design System: https://atlassian.design/design-system
Do not invent components, if a component type is missing but needed find it on the website and install using bun.
Bun as the node package manager.
